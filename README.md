## Commands

The commands must be called from a voice channel.

|                   Command                   |                                      Description                                              |
|:--------------------------------------------|:----------------------------------------------------------------------------------------------|
| `!record`                                   | Start recording the current channel                                                           |
| `!stop`                                     | Stop recording the current channel                                                            |
